# 바당 — Game Design Document

> v1.0 (트랙 A 출품용 확정)  
> 마지막 갱신: 2026-05-21  
> 살아있는 문서. 결정 바뀌면 즉시 업데이트.

---

## 1. 한 줄 정의
**해녀가 되어 바닷속을 탐방하고 정화하는 힐링 게임**

핵심 단어: **정화**, **탐방** (탐험까지는 안 가는, 좁은 맵의 잔잔한 둘러봄)

> "코지"의 정의: **뇌지컬/피지컬 필요 없음** = 점수·콤보·반응속도 같은 스킬 요소 없음. 잘하고 못하고가 없고, 서툰 사람도 좌절 없이 한 판을 완주.  
> 단, "잔잔·명상" 결은 아님. **청소 카타르시스가 중심** (Loddlenaut/Spilled! 결, Abzu 결 아님).

---

## 2. Design Pillars
모든 결정의 기준이 되는 핵심 경험 3개.  
**우선순위**: 충돌 시 위에서 아래로.

1. **줍을수록 맑아지는 쾌감** ⭐ 1순위  
   파워워시 결. 더러움 → 맑음의 즉각적 시각·청각 변화. **청소 카타르시스**. 영역별 진척도. 줍기 자체에 손맛 (드래그 → 망사리).

2. **살아있는 바닷속**  
   산호·여(바위)·생물이 만나고 발견하는 대상. 1번 위에 분위기와 발견을 더함.

3. **스킬 벽 없음**  
   누구나(게임 못해도) 켜서 즐길 수 있음. 실패가 좌절로 안 가야. 코지의 정체성.

> **충돌 예시**: 거북 미니게임 어렵게 → 3번에 막힘. 짧고 직관적.  
> **충돌 예시**: 청소 중 큰 이벤트 → 1번 우선. 배경에서 지나가게.

---

## 3. Player Experience Goals
플레이어가 느꼈으면 하는 감정.

1. **방 청소 끝낸 후련함** — 지저분한 책상 정리하고 한 발 물러서 보는 시원함.
2. **어쩌다 마주친 순간이 기억에 남는** — 그물 묶인 거북 풀어준 순간. 게임 끝나도 떠올라.
3. **피곤한 밤에도 부담 없이 켤 수 있는** — 진도 안 나가도 OK. 5분만 하고 꺼도 만족.

## 4. Core Gameplay Loop

### 4.1 Inner Loop (1회 잠수, ~30초~2분, 등급별)
잠수 → 쓰레기 발견 → **드래그로 망사리에 넣기** → 호흡 떨어지면 수면 ↑ → **숨비소리** → 다시 잠수

### 4.2 Mid Loop (한 세션 = 하루, 5~15분)
잠수 여러 번 → 망사리 가득 또는 만족 → **집으로** = 하루 종료 → 정산 모달 (오늘 줍은 거 + 정화도 변화 + 등급 변화) → 다음 날 시작

### 4.3 Outer Loop (장기)
누적 작업으로 등급 상승 (하군 → 중군 → 상군 → 대상군) → 잠수 가능 시간 늘어남 + 랜턴 시야 넓어짐 + 망사리 용량 증가

> Inner Loop가 가장 중요. 여기서 "줍는 손맛" 안 나오면 위에 쌓는 거 의미 없음.

## 5. 참고 게임 (Inspirations)
무엇을 어떻게 빌리나.

- **Loddlenaut** ⭐⭐ — 가장 가까운 결. 픽셀, 바닷속, 쓰레기 줍기, 생물. **빌리는 것**: 청소 시 산호/식물 색깔 복원, 산소가 위협 안 되게 처리. **안 빌리는 것**: 우주복 정체성, 도구가 너무 많음 (반복성 단점).
- **Spilled!** ⭐⭐ — 사용자 게임과 결 거의 동일. 1시간 코지 청소 게임. **빌리는 것**: 물 맑아지면서 생물 돌아옴, 업그레이드 사이클, "just one more area" 흐름. **안 빌리는 것**: 보트 시점.
- **DrainSim** — 청소 시각적 변화의 카타르시스. 더러움 → 깨끗의 즉각성.
- **PowerWash Simulator** — 1순위 기둥의 정수. before/after 시각 대비, 영역별 진척도.
- **데이브 더 다이브** — 바닷속 풍경 (산호·돌·숨겨진 쓰레기), 잠수/호흡 시스템, 업그레이드 트리 구조. **빌리는 것**: 풍경의 풍성함. **안 빌리는 것**: 전투/식당/스토리.
- **Abzu** — 분위기 자체는 아니지만, 거대 생물 조우의 임팩트 (트랙 B용).

> **공통 단점 (반복성·짧음·깊이 부족)**을 사용자 게임 차별점으로 보완:  
> 한국 해녀 정체성 + 모바일 한 손 + 숨비소리 + 거북 이벤트 통합도.

## 6. 타겟 오디언스
- 스킬 게임 부담스러운 사람
- 모바일로 짬짬이 켜는 사람 (한 손)
- 한국 정서·해녀 문화에 끌리는 사람
- 청소·정리 만족감 좋아하는 사람 (파워워시·정리정돈 콘텐츠 좋아하는 결)

> 코어 게이머 아님. 캐주얼·코지 게이머. 게임 안 하는 친구도 5분 안에 빠질 수 있어야.

## 7. 주요 메커니즘

### 7.1 드래그 줍기 ⭐ 핵심
- 쓰레기 가까이 가서 **드래그 → 손 떼면 "쏙" 망사리로**
- 무게 차등: 가벼운 쓰레기는 빠르게 따라옴, 무거운 폐그물은 천천히
- 바위 뒤/모래 묻힘: 드래그 거리/저항감으로 표현. 끌어내는 느낌
- 즉각 시각·청각 피드백 (반짝, 파티클, "쏙" 사운드)

### 7.2 잠수/호흡/숨비소리
- 호흡 게이지: 잠수 중 줄어듦, 0 되면 강제 수면
- 수면에서 회복 + **숨비소리 사운드** (해녀 정체성 핵심)
- 등급별 잠수 가능 시간: 하군 ~45초 → 대상군 ~2분
- **공기방울 먹기로 잠수 연장** (정화된 산호에서 거품 나옴? — Phase A2에서 디테일)

### 7.3 망사리
- 단순 게이지 (헥사 X — 트랙 B 보류)
- 쓰레기별 차지 무게 다름
- 가득 시 더 줍기 불가 → "집으로" 신호

### 7.4 랜턴
- 해녀 주변만 밝게 보이는 시야
- 등급 따라 넓어짐
- 시야 너머는 어둠 → 발견의 두근거림

### 7.5 등급
- 하군 → 중군 → 상군 → 대상군
- 누적 성과 (정화 수 또는 일차?) 로 자동 승급
- 효과: 잠수 시간 ↑ / 랜턴 범위 ↑ / 망사리 용량 ↑
- **임계값은 한 세션에 1번은 체감되도록**

### 7.6 정화도
- 오늘 맵의 정화 진행 % 상단 표시
- 100% 되면 시각적 보상 (바다 색감, 빛줄기 등)

### 7.7 오염 구역
- 어두운 얼룩 모양의 쓰레기 무리
- 줍는 만큼 얼룩 옅어짐 (영역별 진척도 = 1번 기둥)

### 7.8 여 (바위)
- 물속에 잠긴 바위. 쓰레기 숨길 수 있음

### 7.9 거북 끈 끊기 (서브 이벤트)
- → 11번 참조

## 8. v1 스코프 (공모전용)

**완성해야 할 것** (Phase A1~A5):
- 맵 1개 완성
- 드래그 줍기 (무게 차등)
- 잠수/호흡/숨비소리/공기방울
- 정화도 시각화 (얼룩 옅어짐, 바다 색감 변화)
- 등급 시스템 (하군~대상군)
- 정산 모달
- 거북 서브 이벤트 1개
- 사운드 전반
- 시작/종료 화면

**제외** (트랙 B로):
- 친구 공유, 계정, 리더보드
- 헥사 인벤토리
- 다중 맵 (시간 남으면 추가)
- 소라게 등 추가 서브 이벤트
- 경제·업그레이드 상점

> v1 플레이타임 목표: 1회 5~10분, 전체 클리어 30분~1시간.

## 11. 서브 이벤트 — 거북 끈 끊기
공모전 v1에 포함되는 **유일한** 서브 이벤트.

**시나리오**:
1. 잠수 중 그물 끈에 묶인 거북이 화면에 나타남
2. 거북은 도망감 — 해녀가 따라가서 터치 (도망가는 거북보다 빨라야 함 → 살짝 긴장감)
3. 거북 터치 → 화면 전환 미니게임
4. **가위로 로프 끊기** (드래그 또는 탭 — Phase A4에서 결정)
5. 풀어주면 거북 헤엄쳐 떠남 (사운드·시각 보상)

**디자인 원칙**:
- **소리 중요 / 중독성 있어야 함** — 사용자 강조
- 짧고 직관적, 실패해도 OK (코지 결 — 3번 기둥)
- 보상은 게임 진행보다 **감정·기억**
- 등장 빈도: 너무 자주 X (한 세션에 1번 정도?)

**미정**:
- 끈 끊기 인터랙션 (드래그 가위질? 탭 연타? 라인 따라 그리기?)
- 등장 트리거 (시간? 정화도? 랜덤?)
- 보상 형태 (도감? 단순 사운드+시각? 친밀도 +?)

## 11-B. 서브 이벤트 — 트랙 B 보류
공모전엔 안 들어감. 출품 후 검토.

- **소라게 쓰레기 집 → 진짜 소라 선물**
- **친밀도 시스템** — 누적 플레이 시간/방문 횟수 늘수록 생물들이 안 피함, 가끔 도와달라고 다가옴
- 흘러가는 쓰레기 (시간제한)
- 고래 등 큰 생물 지나가기 (Abzu 결)
